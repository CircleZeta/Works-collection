# 11_ACCEPTANCE_CHECKLIST.md

## v0.1 验收清单

### A. 工程层
- [ ] Gradle Sync 成功
- [ ] `:app` 可以成功编译
- [ ] 应用可以在模拟器或真机启动

### B. 导航层
- [ ] 底部导航显示 4 个入口：首页、趋势、建议、设置
- [ ] 点击后可正常切换页面
- [ ] 页面切换不闪退

### C. 建议页主链路
- [ ] 点击“导入模拟数据”后不报错
- [ ] 点击“生成今日建议”后成功生成结果
- [ ] 建议列表至少显示 1 条以上结果
- [ ] 建议卡片可显示标题、内容、说明、来源

### D. 首页
- [ ] 能读取当日摘要
- [ ] 能显示今日步数
- [ ] 能显示平均心率
- [ ] 能显示睡眠时长
- [ ] 能显示状态摘要文案

### E. 趋势页
- [ ] 能显示近 7 天步数
- [ ] 能显示近 7 天平均心率
- [ ] 能显示近 7 天睡眠时长
- [ ] 无数据时能给出空态提示

### F. 数据层
- [ ] Room 数据库初始化正常
- [ ] `health_metrics` 可写入
- [ ] `final_suggestions` 可写入
- [ ] 重复导入模拟数据不会导致严重失真

### G. 架构层
- [ ] UI 不直接访问 DAO
- [ ] ViewModel 不直接创建数据库
- [ ] Agent 调用通过 `AgentCoordinator`
- [ ] 融合逻辑集中在 `SuggestionFusionEngine`

### H. 非目标确认
- [ ] 当前版本未接入真实蓝牙
- [ ] 当前版本未接入真实远程 Agent
- [ ] 当前版本未实现登录与多用户
- [ ] 当前版本未引入图表库
